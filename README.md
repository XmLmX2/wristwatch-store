## Getting Started

Follow the steps below to get starting with the app.

### Prerequisites

* composer
  ```sh
  composer install
  ```
  
* Symfony migrations
  ```sh
  php bin/console doctrine:migrations:migrate
  ```

* yarn
  ```sh
  yarn install
  yarn encore prod
  ```
